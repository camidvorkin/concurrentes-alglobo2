AlGlobo.com - Process payments

This program sets up N_NODES which will process all of the payments from the payments csv file. It will send the request to all the designated agents, using the configuration in the agents.yaml file.

If the leader node is killed, another one is elected using the ring election agorithm.

Start the program with cargo run --bin alglobo <payments_file>.csv (or default to a csv if not provided)
