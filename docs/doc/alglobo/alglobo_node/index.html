AlgloboNode struct

Handles the logic of each Algobo.com node, where the leader is chosen with the ring election algorithm. The leader will send each transaction result to the replicas.

The communication between nodes is done via UDP, where each node will have two sockets. One for receiving leader election messages and the other for receiving transaction results.

If the replicas reach a timeout without messages from the leader, a new leader is coordinated.
